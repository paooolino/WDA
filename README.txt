1. composer require paooolino/wda:dev-master
2. php vendor/paooolino/wda/generatecode.php vendor/paooolino/wda/tests/samples/sample.ini app
3. cd app
4. composer install
5. point browser >>> done!

You can put vendor/paooolino/WDA/developer_assistant.php file in any directory
(remember to adjust the requires and the loading ini path in the code).
You can use the developer assistant to help keep track of your job.

How to run tests:
composer test
