1. composer require paooolino/wda
2. write config.ini (see example in vendor/paooolino/wda/tests/samples/sample.ini
3. php vendor/paooolino/WDA/generatecode.php config.ini app
4. cd app
5. composer install
6. point browser >>> done!

You can put vendor/paooolino/WDA/developer_assistant.php file in any directory
(remember to adjust the requires and the loading ini path in the code).
You can use the developer assistant to help keep track of your job.

How to run tests:
composer test
