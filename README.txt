1. composer require paooolino/wda:dev-master
2. php vendor/paooolino/wda/generatecode.php vendor/paooolino/wda/tests/samples/sample.ini www
3. cd www
4. composer install
5. use the developer assistant by pointing the www/developer_assistant.php file in browser
6. point browser to www/ >>> done!

You can put vendor/paooolino/WDA/developer_assistant.php file in any directory
(remember to adjust configuration variables in the code).
You can use the developer assistant to help keep track of your job.

How to run tests:
composer test
